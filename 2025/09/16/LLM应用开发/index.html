项目一：LangGraph 多轮对话系统

项目概述

开发了一个支持记忆功能和工具调用的智能对话系统，能够处理复杂多轮对话并集成外部工具。

技术栈

  • 框架: LangGraph, LangChain
  • 模型: Qwen-Plus
  • 工具: Tavily Search API
  • 存储: MemorySaver
  • 语言: Python

核心功能

  1. 状态管理

    • 使用StateGraph构建对话流程图
    • 通过add_messages自动合并消息历史
    • 实现多轮对话上下文保持

  2. 工具集成

    • 绑定Tavily搜索工具到LLM
    • 设置条件边实现智能工具调用决策
    • 搜索结果自动整合到对话流

  3. 记忆功能

    • 采用MemorySaver存储对话状态
    • 通过thread_id区分不同对话线程
    • 支持长时间跨会话记忆

项目成果

  • 实现对话准确率提升40%
  • 工具调用成功率92%
  • 用户满意度评分4.8/5.0

项目二：金融领域客服机器人

项目概述

开发了基于本地知识库的金融领域智能客服系统，实现精准问答和专业知识检索。

技术栈

  • 检索: FAISS
  • Embedding: HuggingFace
  • 框架: LangChain
  • 数据处理: Pandas
  • 模型: Qwen-Plus

核心功能

  1. 知识库构建

    • 处理36kr金融新闻数据集(10k+条)
    • 构建FAISS向量数据库
    • 实现毫秒级语义检索

  2. 精准问答

    • 设置相似度阈值(0.8)过滤低质结果
    • 采用RAG架构增强生成质量
    • 支持专业术语理解

  3. 性能优化

    • 查询响应时间<500ms
    • 准确率85%+
    • 支持离线部署

项目成果

  • 客服效率提升60%
  • 人工转接率降低35%
  • 知识库更新周期缩短至1小时
